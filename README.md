# RustedWarfareDeobfuscation
The deobfuscated game-lib for the Rusted Warfare
## Disclaimer
This repository is not affiliated with the original game developers. The deobfuscated code is provided for educational and research purposes only.
## How to view and contribute
1. Download the latest [Enigma](https://github.com/FabricMC/Enigma).
2. Open the jar file: [game-lib.jar](game-lib.jar)
3. `FIle` -> `Open Mapping` -> `Enigma file` -> select the `mappings.mapping` file.
